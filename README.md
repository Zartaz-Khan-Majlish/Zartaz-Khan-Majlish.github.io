# Zartaz Khan Majlish — Personal Portfolio

## ✨ Features

- **Smooth Scroll Animations** — elements animate in as you scroll using AOS (Animate On Scroll)
- **Tabbed About Section** — Skills, Experience, Education, and Achievements in a clean tab layout
- **Custom Cursor** — dot + ring cursor with hover effects throughout the site
- **Mobile Hamburger Menu** — slide-in drawer navigation for small screens
- **Scroll Progress Bar** — gradient bar at the top tracking read progress
- **Photo Upload** — profile photo can be replaced directly in the browser
- **Responsive Design** — fully responsive across all screen sizes
- **Contact Form** — functional form for direct outreach
- **Resume Download** — one-click PDF download from the nav

## 🛠️ Tech Stack

### Core Languages
- **HTML5** — all structure, content, and semantic markup
- **CSS3** — custom properties, keyframe animations, grid/flexbox layouts, responsive design
- **JavaScript (ES6+)** — all interactivity, tab switching, scroll events, DOM manipulation

### Libraries & Tools
- **[AOS (Animate On Scroll)](https://michaeleskin.com/aos/)** — scroll-triggered entrance animations
- **[Playfair Display](https://fonts.google.com/specimen/Playfair+Display)** — Google Fonts serif display typeface
- **[Formspree](https://formspree.io/)** — backend for the contact form, no server required

### Hosting
- **GitHub Pages** — free static site hosting deployed from the main branch
